# 东方财富股吧数据字段调研结果

## 一、页面 URL 规则

| 页面 | URL 格式 | 示例 |
|------|---------|------|
| 列表页 | `https://guba.eastmoney.com/list,{code},{page}.html` | `/list,600519,1.html` |
| 详情页 | `https://guba.eastmoney.com/news,{code},{post_id}.html` | `/news,600519,1724770672.html` |

## 二、列表页数据字段

每条帖子是 `.listholder .listitem` 或 `#mainlist .listitem`，结构为 table 行：

| 字段 | CSS 选择器 | 示例值 | 说明 |
|------|-----------|--------|------|
| 阅读数 | `div.read` | "2.6万" | 需解析"万"→乘10000 |
| 评论数 | `div.reply` | "291" | |
| 标题 | `div.title a` 或 `div.title a::attr(title)` | "林园：当下是持有茅台以来心情最好的时候" | |
| 帖子链接 | `div.title a::attr(href)` | "/news,600519,1724770672.html" | 拼接域名得完整URL |
| 作者 | `div.author .nametext` | "贵州茅台资讯" | |
| 更新时间 | `div.update` | "06-11 13:53" | 只有月-日 时:分，需推断年份 |

## 三、详情页数据字段

### 3.1 帖子信息

| 字段 | CSS 选择器 | 示例值 | 说明 |
|------|-----------|--------|------|
| 标题 | `.title` 或 `.cn-title` | "林园：当下是持有茅台以来心情最好的时候" | |
| 正文 HTML | `#zw_body` | `<p>【导读】提升分红...` | 包含img/p等HTML标签 |
| 正文纯文本 | `#zw_body` 去标签 | "【导读】提升分红和回购..." | 需 `re.sub(r'<[^>]+>', '', html)` |
| 作者头像 | `.author-info img.avator::attr(src)` | "//avator.eastmoney.com/qface/..." | |
| 作者名 | `.author-info a.name` | "贵州茅台资讯" | |
| 发帖时间 | `.author-info .time` | "2026-06-11 12:40:50" | 完整时间戳 |
| 所属股吧 | `.author-info .bar a` | "贵州茅台吧" | |

### 3.2 评论列表

**⚠️ 评论是 AJAX 延迟加载的！** `domcontentloaded` 时评论 = 0 条，需要等待约 6 秒或 `wait_for_selector(".reply_item")` 后才出现。

评论在 `.reply_item` 中，每条一级评论结构：

| 字段 | ✅ 正确 CSS 选择器 | ❌ 旧选择器（已废弃） | 示例值 |
|------|-------------------|---------------------|--------|
| 评论ID | `::attr(data-reply_id)` | — | "9906290464" |
| 作者 | **`.item_reuser a::text`** | ~~`.item_reuser::text`~~ | "勿谓言之不預" |
| 评论时间 | **`.publishtime .pubtime::text`** | ~~`.publishtime::text`~~ | "2026-06-11 23:38:41" |
| 来源地域 | **`.publishtime .ipfrom::text`** | ~~split("来自")~~ | "来自 上海" → 需去掉"来自 " |
| 评论内容 | **`.reply_title_span::text`** | ~~`.reply_title::text`~~ | "技术含量也就一瓶酒..." |
| 点赞数 | ⚠️ 初始评论无点赞数字 | ~~`.reply_bottom`找数字~~ | 默认 0 |
| 子评论 | `.reuser_l2` | — | 二级评论，格式"作者：内容" |

## 四、数据模型对应关系

### 列表页 → Post 列表抓取

```
列表页 → 遍历 .listitem → 提取 title/read_count/comment_count/author/update_time/post_url
       → 入库（帖子基础信息）
       → 进入详情页 → 提取 content + comments
```

### 详情页 → 正文 + 评论

```
详情页 → #zw_body 提取正文（HTML + 纯文本）
       → .reply_item 遍历评论（author/time/content/likes/reply_count/sub_comments）
```

---

## 五、爬取策略

### 5.1 流程

```
1. 列表页翻页 → 收集帖子URL列表
2. 逐个访问帖子详情页 → 提取正文 + 评论
3. 存入数据库
```

### 5.2 翻页规则

URL: `/list,{code},{page}.html`
- page 从 1 开始递增
- 每页约 20 条帖子
- 约到 660 页后可能被限制（需重启浏览器）

### 5.3 反爬注意

- **IP 黑名单**：请求过快会触发，建议每页间隔 1~2 秒
- **验证码**：登录后访问可减少触发
- **图片加载**：不要禁用图片加载（会触发黑名单）

### 5.4 cookie 使用

用户会提供登录后的 cookie，在 Playwright 中注入：

```python
await context.add_cookies([
    {"name": "cookie_name", "value": "cookie_value", "domain": ".eastmoney.com", "path": "/"}
])
```
