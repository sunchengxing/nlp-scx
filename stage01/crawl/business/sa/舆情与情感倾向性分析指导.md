# 舆情与情感倾向性分析 — 任务指导文档

## 一、任务定义

| 项目 | 说明 |
|------|------|
| **任务名称** | 论坛信息抓取（非结构化数据）→ 舆情与情感倾向性分析 |
| **目标** | 从东方财富股吧等论坛抓取帖子，训练情感分类模型，判断舆情倾向 |
| **核心问题** | 给定一条论坛帖子标题，判断其对股票走势的态度：看多 / 看空 / 中性 |

---

## 二、Baseline 数据集

### 2.1 数据集信息

| 项目 | 详情 |
|------|------|
| **名称** | `HikasaHana/eastmoney_guba_title` |
| **来源** | 东方财富 上证50ETF 股吧帖子标题 |
| **地址** | https://huggingface.co/datasets/HikasaHana/eastmoney_guba_title |
| **规模** | 10,646 条（train: 8,520 / test: 2,130） |
| **大小** | 637 kB |
| **语言** | 中文 |
| **格式** | CSV（另有自动转换的 Parquet） |
| **标注方式** | 人工标注，三元分类 |

### 2.2 字段说明

| 字段 | 类型 | 范围 | 说明 |
|------|------|------|------|
| `title` | string | 1–79 字 | 股吧帖子标题（中文） |
| `label` | int64 | 0 / 1 / 2 | 情感标签 |

### 2.3 标签定义（⚠️ 和常见情感分类不同，需特别注意）

| label | 含义 | 说明 |
|-------|------|------|
| 0 | **negative（看空）** | 对股票上涨持消极态度 / 对股票下跌持积极态度 |
| 1 | **positive（看多）** | 对股票上涨持积极态度 / 对股票下跌持消极态度 |
| 2 | **neutral（中性）** | 无主观倾向的提问、与走势无关的内容、多空摇摆的观点 |

> ⚠️ **关键理解**：这里的 positive/negative 不是"开心/难过"，而是"看涨/看跌"。
> 这是**金融领域特有的情感定义**，和通用情感分析不同。
> 例："暴跌好！" 在通用情感里是 positive，但在金融情感里是 **negative（看空）**。

### 2.4 数据样例

| title | label | 解读 |
|-------|-------|------|
| "弱鸡。免强拉起来了的大盘能不能持久呢？十月3100都可能要破！" | 0 | 看空：认为大盘撑不住 |
| "没有戏了" | 0 | 看空：悲观 |
| "不要怀疑趋势，继续逢低做多，中级上升行情打开窗口" | 1 | 看多：明确看涨 |
| "下周看涨" | 1 | 看多：直接预测涨 |
| "请问ETF交易费用是多少，能T+0吗？" | 2 | 中性：纯提问，无倾向 |
| "这个能不能当股票买卖？怎么那么多人谈期权？" | 2 | 中性：疑问，无立场 |

### 2.5 数据质量注意事项

- 标注者自述偶尔会困惑，建议**重新筛查**标注
- 原作者观点："自己标注的才是最信得过的"
- 自动转换的 Parquet 分支可能过时，推荐用 Python 自行转换
- 数据仅覆盖 **上证50ETF 单一股吧**，领域较窄

## 五、阶段2：模型训练

### 5.1 预训练模型选择

金融中文情感分析，推荐优先级：

| 优先级 | 模型 | 理由 |
|--------|------|------|
| ⭐⭐⭐ | `chinese-bert-wwm-ext` | 中文全词掩码，金融文本基础表现好 |
| ⭐⭐⭐ | `FinBERT`（如有中文版） | 金融领域专用预训练 |
| ⭐⭐ | `chinese-roberta-wwm-ext` | 比 BERT-wwm 略强，但非金融专用 |
| ⭐ | `MacBERT` | 通用中文，金融上未必更优 |

> 数据集作者用 `HikasaHana/BERT-WMM` 训练了 baseline 模型，可作参考。

### 5.2 训练流程

```
1. 加载预训练模型 + tokenizer
2. 对 title 做 tokenize（max_length 建议先看长度分布再定）
3. 划分 train/val/test（原数据集已有划分，可再加 val）
4. 设置训练参数：
   - epochs: 3~5（小数据集容易过拟合，先从 3 开始）
   - batch_size: 16~32
   - learning_rate: 2e-5 ~ 5e-5
   - weight_decay: 0.01
5. 训练 → 评估 → 调参
```

### 5.3 评估指标

**不要只看 accuracy**，金融场景下：

| 指标 | 为什么重要 |
|------|-----------|
| **F1-macro** | 三分类均衡性，各标签表现是否均衡 |
| **Precision(negative)** | 看空的精确率 — 误判看空会导致错误决策 |
| **Recall(negative)** | 看空的召回率 — 漏掉看空信号会错过风险 |
| **混淆矩阵** | 看多/看空/中性之间的误判模式 |

### 5.4 对比实验建议

- [ ] 不同预训练模型对比
- [ ] 有无数据增强对比
- [ ] 不同 max_length 对比
- [ ] 2分类（去中性） vs 3分类 对比

## 八、关键风险与注意事项

| 风险 | 说明 | 应对 |
|------|------|------|
| 数据量偏少 | 1万条训练分类器，过拟合风险高 | 数据增强 + 正则化 + 早停 |
| 标注质量 | 原作者自述有困惑 | 必须抽样复查 |
| 领域狭窄 | 仅上证50ETF股吧 | 补充其他股吧数据 |
| 标签定义反直觉 | 金融情感 ≠ 通用情感 | 明确文档，代码中用枚举不要硬编码数字 |
| 标题信息有限 | 只有标题，没有正文 | 后续抓取时补充 content 字段 |
| 概念漂移 | 股市语境随时间变化 | 定期重新标注 + 重训练 |
