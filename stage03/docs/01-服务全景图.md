## 1. 语音识别服务全景图

科大讯飞提供 **4 大类在线语音识别 API** + **3 大类离线 SDK**：

| 服务名称 | 接入方式 | 音频时长 | 适用场景 |
|---------|----------|---------|---------|
| **语音听写（流式 IAT）** | WebSocket API | ≤ 60s | 短语音实时转写，边录边识别 |
| **大模型语音识别（Spark IAT）** | WebSocket API | ≤ 60s | 升级版，支持 202 种方言免切换 |
| **实时语音转写（RTASR）** | WebSocket API | 无限制 | 会议、直播等连续长音频实时转写 |
| **语音转写（Lfasr）** | REST API | ≤ 5h | 离线长音频异步转写，提交→查询→获取 |
| **离线语音听写** | SDK（Android 等） | - | 无网络环境下的语音识别 |
| **离线命令词识别** | SDK（多平台） | - | 特定命令词匹配，如智能家居控制 |
| **语音唤醒** | SDK（多平台） | - | 设备端关键词唤醒 |
