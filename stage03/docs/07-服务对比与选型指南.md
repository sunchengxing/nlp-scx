## 鉴权机制对比

### HMAC-SHA256 系列（IAT / Spark IAT）

```
# 1. 签名字符串
"host: $host\ndate: $rfc1123_date\n$request_line"

# 2. 用 APISecret 做 HMAC-SHA256 签名，Base64
signature = base64(hmac_sha256(signature_origin, APISecret))

# 3. 拼接并再次 Base64 得到 authorization
auth_origin = 'api_key="$api_key",algorithm="hmac-sha256",headers="host date request-line",signature="$signature"'
authorization = base64(auth_origin)
```

### HMAC-SHA1 系列（RTASR / Lfasr）

```
# 统一公式
signa = base64(hmac_sha1(md5(app_id + ts), secret_key))
```

## 选型决策树

```
需要实时返回？
├── 是 → 音频时长？
│   ├── ≤60s 短语音 → 方言多/效果要求高？
│   │   ├── 是 → Spark IAT（202种方言免切换）
│   │   └── 否 → IAT 流式听写（成熟稳定）
│   └── >60s / 无限制 → RTASR 实时转写
└── 否 → 长音频离线处理 → Lfasr 语音转写
         （录音文件，5小时以内，异步提交+轮询获取）
```

## 费用相关
- 所有服务提供**免费试用额度**（注册+实名认证后）
- 正式商用需购买服务量（按时长/次数计费）
- 离线 SDK 按装机量购买授权
- 方言、小语种等特殊能力需单独授权开通
